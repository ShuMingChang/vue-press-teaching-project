# vue-press-teaching-project

此專案是用來介紹vuepress相關內容的筆記
哪容關鍵字如下：
- Jamstack
- Cloudflare Pages
- Markdown

[內容網站](https://vue-press-teaching-project.pages.dev/)

## Project setup
```
cd docs
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
